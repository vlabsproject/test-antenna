Red Box - Transmitter

Green Box - Receiver

1. Move the receiver box to the desired location by dragging it with the mouse.
2. Enter the required parameters in the input fields.
3. Click the "Calculate" button to calculate the signal strength at the current location of the receiver box.
4. Click the "Take Reading" button to capture the current values and signal strength and add them to the readings table.
5. Repeat steps 1 to 4 to capture multiple readings.
